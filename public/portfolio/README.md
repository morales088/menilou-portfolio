## Image Guidelines

- **Format:** JPG, PNG, or WebP
- **Recommended size:** 1200x1200px (square) for gallery images
- **Cover images:** 1600x900px (16:9 aspect ratio)
- **File size:** Keep under 500KB for optimal performance
- **Naming:** Use lowercase, hyphens for spaces

## Adding New Images

1. Place images in the appropriate client folder
2. Update the `clientsData.ts` file if you add new media items
3. Ensure file names match those referenced in the data file

## Video Support

Videos are also supported. Use `.mp4` format and keep file sizes reasonable (<10MB recommended).
